This project is a responsive Login and Registration user interface developed as part of Internship Task-2.  
It is built using HTML, CSS, JavaScript, and Bootstrap to demonstrate core frontend development concepts.  
The application provides separate login and registration pages with a clean and user-friendly interface.  
Client-side form validation is implemented to ensure proper user input.  
A show and hide password feature improves usability and user experience.  
Basic interactive behavior is added using JavaScript.  
Bootstrap is used to create a responsive and mobile-friendly layout.  
The project follows beginner-friendly coding practices and clear file structure.  
No backend or database integration is included in this implementation.  
This project is intended for learning, practice, and internship demonstration purposes.
