This project is a responsive Login and Registration user interface developed as part of Internship Task-2. It is built using HTML, CSS, JavaScript, and Bootstrap to demonstrate core frontend development concepts. The application provides separate login and registration pages with a clean and user-friendly interface, along with client-side form validation to ensure proper user input. A show and hide password feature is included to enhance usability and user experience, while basic interactive behavior is implemented using JavaScript. Bootstrap is used to create a responsive and mobile-friendly layout. The project follows beginner-friendly coding practices with a clear file structure and does not include any backend or database integration. It is intended for learning, practice, and internship demonstration purposes.
